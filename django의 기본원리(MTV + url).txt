django의 기본원리(MTV + url)

Model 
Templete : 프론트 - 1. html : 화면의 구성(기본)  이것만
                               2. css : 디자인
                               3. js : 동적인 부분
View


- Template 기초 (html)
<html> = 시작과 끝
<head> : 파일의 정보 (보이지 않는 부분)
<body> : 화면에 표시되는 부분

-<div>태그 : 구역나타내기
-<h1> ~ <h6> : 제목출력
-<p> : 문단 출력
-<br> : 줄바꿈
-<img> : 이미지 삽입 , scr속성에 이미지의 경로를 입력


-<a> : 하이퍼링크를 걸기 위한 태그, href속성에 주소 명시

-<form>  : 서버로 정보를 제출하기 위한 태그   -- 양식
 -- action : 제출 url
  -- method : 제출할 방법(GET - 누구나 접근, POST - 숨겨져서 전달)
-<input> : 사용자에게 정보를 입력 받기 위한 태그
